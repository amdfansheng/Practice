# Practice

double linked list support features:
  1. support any item type.
  2. iterator operator ++, --, *, ->
  3. scope for loop
  4. initialization with list
  5. support operation:
  
    a. pop element
    b. push element
    c. insert element
    d. remove value
    e. erase item at special position
    f. find element that has special value
